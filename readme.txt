 Assignment: Build Your Personal Profile Card

 CSS Types Used:
1. Inline CSS:
   - Used in <div class="profile-card"> to set background-color.

2. Internal CSS:
   - Inside <style> tag in the <head>.
   - Styles applied to h1, h2 and descendant selector for .contact p.

3. External CSS:
   - Linked via style.css.
   - Contains most layout and styling including selectors.

 CSS Selectors Used:
1. ID Selector: 
   - #profile-pic (styles profile image)

2. Class Selector:
   - .bio, .contact, .profile-card

3. Element Selector:
   - body, h1, p, ul, li

4. Group Selector:
   - h1, h2

5. Descendant Selector:
   - .contact p, .contact a

6. Attribute Selector:
   - a[href] for styling all anchor tags with href
